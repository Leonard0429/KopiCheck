# KopiCheck

KopiCheck is a static product website for an AI scam-checking Telegram bot designed mainly for elderly users aged 60 and above in Singapore, with adults aged 18 to 59 as secondary users.

The bot helps users check suspicious SMS, emails, and links before they click, share sensitive information, or make unsafe online decisions. The project uses Telegram, n8n, an AI Agent, OpenRouter/DeepSeek, memory, Google Sheets knowledge base, and Google Sheets testing log.

## Pages Included

- `index.html` - Home page and product story
- `checker.html` - Telegram bot usage steps and rule-based demo simulator
- `guide.html` - Scam warning signs, fake message examples, and safe actions
- `how-it-works.html` - Technical workflow, stack, screenshots, and project evidence
- `about.html` - Project purpose, target audience, team contributions, and disclaimer

## How To Run Locally

Open `index.html` directly in a browser.

No backend server, API keys, database, or paid libraries are required. The website uses only HTML, CSS, and JavaScript.

## GitHub Pages Deployment Note

This project is ready for GitHub Pages because all page links and asset paths are relative.

To deploy later:

1. Push the project folder to a GitHub repository.
2. Go to repository `Settings`.
3. Open `Pages`.
4. Select the branch and root folder.
5. Save and wait for GitHub Pages to publish the site.

## Disclaimer

KopiCheck provides scam awareness support and does not replace official advice from banks, Singpass, government agencies, or law enforcement.

The website demo uses simple rule-based JavaScript for illustration. The full AI analysis happens in the KopiCheck Telegram bot powered by n8n and AI.
